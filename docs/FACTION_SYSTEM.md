# Faction System

The world should contain more than sects. Use a mix of fixed legacy powers and generated powers.

## Fixed Legacy Powers

- Great Xia Dynasty
- Imperial Academy
- Divine Hall
- Wudang Sect
- Shaolin Temple
- Tang Clan of Shu
- Mount Hua Sword Sect
- Medicine King Valley
- Thousand Mechanism Workshop
- Beggar Sect
- Black Sun Cult
- Abyssal Remnant

## Institution Types

Use Sect, Clan, Temple, Monastery, Academy, Palace, Tower, Valley, Alliance, Cult, Workshop, Manor, Dynasty, and Hidden Lineage.

## Alignments

- Righteous Path
- Noble Clan
- Imperial Order
- Academy
- Divine Hall / Temple
- Rogue Path
- Heretical Path
- Abyssal Path
- Hidden Lineage
