# Roadmap

1. Split the single-file prototype into data, entities, generators, simulation, observer, UI, styles, and utils folders.
2. Add public record and true record variants for major chronicle events.
3. Add faction-specific event tables for the Tang Clan, Imperial Academy, Divine Hall, Black Sun Cult, and Abyssal Remnant.
4. Add paths, qi types, techniques, and faction weaknesses as first-class data.
5. Add causality tracing so users can inspect why wars, betrayals, inheritances, and calamities happened.
