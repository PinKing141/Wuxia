# Cultivation System

The simulator uses one universal realm ladder for all factions and paths.

| Rank | Realm |
| ---: | --- |
| 0 | Mortal |
| 1 | Body Tempering |
| 2 | Meridian Opening |
| 3 | Qi Refining |
| 4 | Foundation Establishment |
| 5 | Golden Core |
| 6 | Nascent Soul |
| 7 | Soul Formation |
| 8 | Void Refinement |
| 9 | Body Integration |
| 10 | Mahayana |
| 11 | Tribulation Transcendence |
| 12 | Immortal / Ascended |

Sub-stages should generally be Early, Middle, Late, Peak, and Half-Step when the simulator expands beyond the current prototype.

Cultivation paths are expressions of the same ladder: Martial, Sword, Saber, Daoist, Buddhist, Scholarly, Heretical, Poison, Medicine, Formation, Talisman, Ghost/Corpse, Assassin, Beast-Taming, and Artifact Refining.
